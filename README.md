# Lexicon Todos Prisma

## Getting Started

```bash
# Clone the repository
git clone https://github.com/LexiconAlex/lexicon-todos-primsa.git

# Navigate to project directory
cd lexicon-todos-primsa
```

## Installation
```bash
npm install
```

## Scripts
```bash
 # Compile TypeScript and run app.js
npm start 
```
